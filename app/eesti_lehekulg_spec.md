# Эстонский — личная база знаний

## Цель проекта

Одностраничный HTML-сайт с моими заметками по эстонскому языку. Открывается с телефона и с компа. Тёмная тема, минималистичный дизайн в стиле Anthropic Claude. Это первая итерация — будет расти со временем (буду добавлять новые таблицы, правила, заметки по управлению глаголов).

## Технические требования

- Один HTML-файл (`index.html`), inline CSS и JS внутри
- Никаких внешних зависимостей (никаких CDN, фреймворков)
- Полностью мобильная адаптация: viewport meta, гибкая сетка
- Тёмная тема — единственная, без переключателя
- Шрифт: системный stack — `-apple-system, BlinkMacSystemFont, "SF Pro Text", "Segoe UI", system-ui, sans-serif`
- Класть в существующий проект `/Users/shabak/advertum/advertum`. Имя файла обсудить — возможно `eesti.html` или папка `eesti/index.html`.

## Цветовая палитра

Тёмная тема, тёплая (не холодно-синяя, а серо-бежевая, как у Anthropic):

```css
:root {
  --bg: #262624;            /* фон страницы — тёплый тёмно-серый */
  --surface: #30302e;       /* фон блоков и таблиц */
  --surface-2: #3a3a37;     /* hover / лёгкое выделение */
  --text: #faf9f5;          /* основной текст — тёплый белый */
  --text-dim: #b6b5ad;      /* вторичный текст / подписи */
  --border: rgba(250, 249, 245, 0.08); /* тонкие границы */

  --purple-bg: rgba(196, 168, 232, 0.12);
  --purple-fg: #c4a8e8;
  --purple-border: rgba(196, 168, 232, 0.25);

  --teal-bg: rgba(128, 196, 190, 0.12);
  --teal-fg: #80c4be;
  --teal-border: rgba(128, 196, 190, 0.25);

  --amber-bg: rgba(232, 195, 120, 0.12);
  --amber-fg: #e8c378;
  --amber-border: rgba(232, 195, 120, 0.25);
}
```

Если эти hex-значения не выглядят достаточно «по-антропиковски» в твоей реализации — подбери на глаз ближе к ощущению `claude.ai` тёмной темы. Главное: **тёплый, не холодный**; **мягкие пастельные акценты**, не насыщенные неоновые.

## Типографика

- Заголовки страницы: 24-28px, weight 600, цвет `--text`
- Подзаголовки секций: 18-20px, weight 500, цвет `--text`
- Подписи / пояснения: 13-14px, weight 400, цвет `--text-dim`
- Текст в таблицах: 14-15px на десктопе, 13-14px на мобиле
- Эстонские слова: можно чуть жирнее (weight 500) для читаемости

## Структура страницы

### Шапка

- Заголовок: «Эстонский»
- Подзаголовок: «Личная база знаний» (или подобное — на твой вкус)

### Секция 1: Вопросительные слова — три категории

Три колонки в горизонтальном гриде на десктопе. На мобиле — три блока друг под другом (колонки складываются вертикально при ширине экрана < 720px).

Заголовок секции: «Вопросительные слова — три группы»

Подзаголовок: «У каждой группы своё поведение»

#### Группа 1 (фиолетовая) — Склоняемые

Заголовок группы: «Склоняемые»
Описание под заголовком: «меняют форму по падежу»

Карточки внутри группы:

| Слово | Перевод |
|-------|---------|
| kes? | кто? (о людях) |
| mis? | что? (о предметах) |
| milline? | какой? (выбор из вариантов) |
| missugune? | какой? (общее качество) |

#### Группа 2 (бирюзовая) — О месте

Заголовок: «О месте»
Описание: «фиксированная тройка»

| Слово | Перевод |
|-------|---------|
| kus? | где? |
| kuhu? | куда? |
| kust? | откуда? |

#### Группа 3 (янтарная) — Неизменяемые

Заголовок: «Неизменяемые»
Описание: «одна форма»

| Слово | Перевод |
|-------|---------|
| millal? | когда? |
| kui kaua? | как долго? |
| kuidas? | как? |
| kui palju? | сколько? (неисчисляемое) |
| mitu? | сколько? (исчисляемое) |
| miks? | почему? |
| kas? | ли? (да/нет) |

**Стиль карточек:** скруглённые блоки (radius 8px), фон `--{color}-bg`, обводка 1px `--{color}-border`, padding ~12-14px. Эстонское слово сверху (weight 500, цвет `--{color}-fg`), русский перевод под ним мельче (цвет `--text-dim`).

### Секция 2: Склонение четырёх «фиолетовых» слов

Заголовок: «Склонение по падежам»
Подзаголовок: «8 главных форм для каждого из 4 склоняемых вопросов»

Таблица — на десктопе обычная, на мобиле горизонтальный скролл (overflow-x: auto в обёртке).

Структура таблицы:

| Падеж | Вопрос | kes | mis | milline | missugune |
|-------|--------|-----|-----|---------|-----------|
| nimetav | кто? что? какой? | kes | mis | milline | missugune |
| omastav | кого? чей? | kelle | mille | millise | missuguse |
| osastav | кого? что? (вин.) | keda | mida | millist | missugust |
| alalütlev | у кого? у чего? | kellel | millel | millisel | missugusel |
| allatiiv | кому? чему? | kellele | millele | millisele | missugusele |
| alaltütlev | от кого? от чего? | kellelt | millelt | milliselt | missuguselt |
| kaasaütlev | с кем? с чем? | kellega | millega | millisega | missugusega |
| ilmaütlev | без кого? без чего? | kelleta | milleta | milliseta | missuguseta |

**Стиль таблицы:**
- Вся таблица «фиолетового» цвета (фон `--purple-bg`, обводка `--purple-border`)
- Заголовки колонок (kes/mis/milline/missugune) и ряд `nimetav` выделены чуть ярче — например, цветом `--purple-fg` для самих слов
- Колонка «Падеж» — название падежа (weight 500, цвет `--text`) и под ним русский вопрос (weight 400, цвет `--text-dim`, чуть мельче)
- Между ячейками тонкие линии 1px цвета `--purple-border`
- Ячейки с эстонскими формами — выровнены по центру, padding 10-14px, шрифт 14-15px

### Дизайн-принципы (важно)

1. **Воздух важнее декора.** Большие отступы между секциями (60-80px на десктопе, 40px на мобиле). Не забивать страницу контентом плотно.
2. **Минимум границ.** Полагаться на цветной фон карточек, а не на жирные обводки.
3. **Радиусы скругления** ~8px на карточках, ~12-16px на больших контейнерах.
4. **Никаких теней-блюров.** Тёмный фон + цветные плоскости работают сами по себе.
5. **Никаких иконок и декоративных элементов** в этой версии. Только текст и цвет.

### Адаптивность

- Брейкпоинт ~720px: на меньших экранах три колонки складываются в одну (3 ряда)
- Таблица склонений на мобиле — горизонтальный скролл с зафиксированной первой колонкой (sticky), либо без фиксации (просто скролл)
- Минимальная ширина шрифта на мобиле: 13px для тела, 11px для подписей
- Padding контейнера: 16-20px на мобиле, 32-48px на десктопе
